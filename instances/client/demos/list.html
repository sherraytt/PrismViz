输入数据需求

List 是独立右侧信息块，输入什么就展示什么。它不关心 Scroll、Prism 或 Tree。

{
  items: [
    { id, title, subtitle, value, color, raw }
  ]
}

也可以直接输入任意对象，以 JSON / table 形式展示：

{
  id: "ent_001",
  name: "企业 A",
  score: 0.857142857
}

数字默认最多保留 4 位小数。点击左侧条目只更新本 demo 外层传给右侧 List 的数据。
