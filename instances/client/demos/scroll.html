输入数据需求

Scroll 消费一个 slice 的图数据，独立绘制卷轴图。客户数据默认使用 free graph 布局，并通过同一套 renderScroll() 入口渲染。

{
  slice: { id, name, shortName, color },
  graph: {
    nodes: [
      { id, label, primarySliceId, sliceWeights, time, impact, meta }
    ],
    edges: [
      { source, target, weight, label }
    ],
    contextEdges: [
      { source, target, direction, sourceSliceId, targetSliceId, weight, time }
    ]
  },
  slices: [
    { id, name, shortName, color }
  ],
  colorMap: {
    [sliceId]: color
  }
}

单击 Scroll 中的节点或边后，会在这里用 List 组件展示固定详情；点右上角 × 回到输入说明。
