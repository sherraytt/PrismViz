输入数据需求

Prism 消费 active slices、slice 内部图，以及 slice 间聚合矩阵。客户数据默认无时间分层，因此默认使用 free graph，但仍通过同一套 renderPrism() 入口渲染。

{
  slices: [
    { id, name, shortName, chordName, color, size }
  ],
  sliceGraphs: {
    [sliceId]: {
      nodes: [
        { id, label, primarySliceId, time, impact }
      ],
      edges: [
        { source, target, weight }
      ]
    }
  },
  chord: {
    slices,
    matrix: [
      [0, 3, 2],
      [1, 0, 4],
      [5, 2, 0]
    ]
  },
  colorMap: {
    [sliceId]: color
  }
}
