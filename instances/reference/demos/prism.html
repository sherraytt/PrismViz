Prism

输入数据需求

Prism 消费 active slices、slice 内部图，以及 slice 间聚合矩阵。组件内部负责把输入数据绘制成柱体、柱面缩略图、上方 Chord 盖和下方封底。

{
  slices: [
    { id, name, shortName, chordName, color, size }
  ],
  sliceGraphs: {
    [sliceId]: {
      nodes: [
        { id, label, primarySliceId, time, impact }
      ],
      edges: [
        { source, target, weight }
      ]
    }
  },
  chord: {
    slices,
    matrix: [
      [0, 3, 2],
      [1, 0, 4],
      [5, 2, 0]
    ]
  },
  colorMap: {
    [sliceId]: color
  }
}

当前 demo

original weighted 默认复刻原版 drawGeneticPrism() 的柱面宽度规则：先取关系矩阵行和，再执行 adjustWeight 保底，最后用权重对应角度算每个柱面的弦长。Prism 会按容器实际尺寸推导柱体半径、高度和盖板厚度。

equal width 使用等权重，便于对比和后续业务场景固定版式。time layered 让 Slice 缩略图使用时间层次，free graph 则不依赖时间字段。Prism 内部保留自动旋转、拖动旋转、滚轮缩放和标签 hover bubble；这些交互不触发其它组件。
