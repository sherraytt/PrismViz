Chord

输入数据需求

Chord 消费 slice 定义和 slice 间聚合关系矩阵，绘制原来左侧 Chord Diagram 的环形弦图。

{
  slices: [
    { id, name, shortName, chordName, color, size }
  ],
  matrix: [
    [0, 3, 2],
    [1, 0, 4],
    [5, 2, 0]
  ]
}

其中 matrix[i][j] 表示第 i 个 slice 到第 j 个 slice 的聚合关系数量。组件内部只做自身 hover 高亮，不切换 Scroll 或 Prism。
