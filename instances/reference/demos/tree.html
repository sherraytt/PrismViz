输入数据需求

Tree 默认消费通用层级数据，也可以直接用 slice 列表生成扁平目录。甲方已有产业图谱时，可以直接传入外部 hierarchy。

{
  slices: [
    { id, name, shortName, color, size }
  ]
}

如果需要层级形式，也可以输入：

{
  hierarchy: [
    { id, parentId, level, name, selectable, count, color }
  ]
}

本 demo 使用 model.tree，展示外部 hierarchy 归一化后的切片目录。
